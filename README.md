# Customer Profitability Analysis — Pareto Principle

## Problem
An e-commerce store assumed all customers contribute 
equally to profit. The data revealed a very different reality.

## Key Findings
- A small customer segment (~10-20%) generates 
  the majority of total profit
- A significant portion of customers show unprofitable 
  patterns, mainly due to high return rates and low 
  order frequency
- Top customers are almost entirely returning users 
  with high purchase frequency and low return rates
- Low-value customers show higher return rates 
  and fewer repeat purchases

## What Makes a Top Customer?
1. Returning customer (not new)
2. High order frequency
3. Low return rate

## Tools
Python · Pandas · Matplotlib · Seaborn

## ⚠️ Data Note
Synthetic dataset — patterns are directionally valid.
Exact percentages may differ in real-world scenarios.

## Recommendation
- Prioritize retention of high-value customers
- Improve acquisition quality, not just volume
- Reduce return rates, especially in low-value segments
